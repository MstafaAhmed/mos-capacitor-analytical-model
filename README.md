# MOS Capacitor Analytical Model in MATLAB

This repository contains MATLAB implementations of the analytical electrostatic equations of an ideal MOS capacitor.

The project demonstrates the relationship between gate voltage, surface potential, semiconductor charge, oxide thickness, and charge components using the exact analytical formulation commonly presented in semiconductor device textbooks.

## Features
Exact semiconductor charge calculation
Gate voltage vs. surface potential
Semiconductor charge vs. gate voltage
Semiconductor charge vs. surface potential
Oxide thickness comparison
Accumulation, depletion, and inversion regions
MATLAB implementation using analytical equations
Equations
